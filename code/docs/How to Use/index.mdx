## Launching the Tool
First, be sure that you've [installed the Tool](/install). Then, just like a bookmark to a website, click the icon for the "LibGuides Accessibility Tool" in your bookmark bar to activate the tool! After a moment, the Tool will appear as a sidebar on the page.
