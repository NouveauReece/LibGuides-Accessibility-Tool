## Can I use your source code for my own projects?
Sure thing! Our source code is available under the MIT License, so feel free [fork our repository on GitHub](https://github.com/NouveauReece/LibGuides-Accessibility-Tool) and continue using it to make the web a more accessible place!  

Note that LAT utilises logic from [axe-core by Deque Systems](https://github.com/dequelabs/axe-core), which is available under the Mozilla Public License, version 2.0 (MPL-2.0). If you make modifications to our code, please ensure you are still using code from axe-core in a manner permitted under MPL-2.0.
